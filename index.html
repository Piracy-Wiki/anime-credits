A list of copyright text ready to be pasted in anything that needs attribution.

I took up a list of copyright credit text as part of my blogging stint, but now I want to expand it from a side-project to something of a web application.

Not all information here is accurate, so I need everyone's help and cooperation. Let's make this side-project better! You can pull the Github repo and check out the controller.js which contains the database itself. If you are not into Git, you can request titles by tweeting me.

This side-project is made with Bootstrap and Angular.JS 1.3.6 (which they have phased out in favor of Angular.io)
